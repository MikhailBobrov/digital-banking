**Для запуска и сборки проекта необходимо:**
* npm install
* npm run sass
* в package.json прописать в scripts: *"scripts": {
  "sass": "node-sass -w scss/ -o dist/css --recursive"
  },*
* для создания карусели использован jQuery плагин - Slick slider
